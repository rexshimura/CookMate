# CookMate App - NPM Dependencies List

## Frontend Dependencies (install in frontend folder)
```bash
npm install react react-dom react-router-dom tailwindcss @tailwindcss/vite lucide-react axios firebase
```

## Backend Dependencies (install in backend/functions folder)
```bash
npm install express firebase-admin firebase-functions dotenv axios cors bcryptjs jsonwebtoken @google/generative-ai openai replicate langchain joi
```

## Root Dependencies (install in root folder)
```bash
npm install concurrently
```

---

**Installation Commands:**
```bash
# Root level
npm install

# Frontend
cd frontend && npm install

# Backend
cd backend/functions && npm install
```

**Total packages needed:** 20+ core dependencies
